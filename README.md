## HTML Attributes

To harness CoCreateJS's capabilities, you can use the following HTML attributes in your web app:

-   **Socket HTML Attributes:** ['broadcast', 'broadcast-sender', 'broadcast-browser', 'namespace', 'room', 'balancer']

-   **CRUD HTML Attributes:** ['storage', 'database', 'array', 'object', 'key', 'index', 'save', 'read', 'update', 'delete', 'realtime', 'crud', 'upsert', 'value-type', 'value-prefix', 'value-suffix']

-   **Filter HTML Attributes:** ['filter-selector', 'filter-closest', 'filter-parent', 'filter-previous', 'filter-next', 'filter-key', 'filter-value', 'filter-value-type', 'filter-case-sensitive', 'filter-operator', 'filter-logical-operator', 'filter-sort-key', 'filter-sort-direction', 'filter-search', 'filter-limit', 'filter-count', 'filter-on']

-   **Render HTML Attributes:** ['render', 'render-selector', 'render-closest', 'render-parent', 'render-previous', 'render-next', 'render-as']

By incorporating these attributes in your HTML, you can fully leverage the power of CoCreateJS for your web app.
